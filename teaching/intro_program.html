Course Introduction

Computers, including PC, mobile phones, etc, are increasingly becoming an indispensable device in our daily life, where the most essential part of a computer is the software, more specifically, applications or apps. In the new era, learning or understanding programming is not only for software engineers, but also for people who want to improve his/her skillset. The purpose of this course is to help students to understand what the programming is and how to write a simple application. In this course, we will use a very popular programming language, python. After this course you may take other advanced programming courses to further improve your programming skills.
